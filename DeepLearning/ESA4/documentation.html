Beschreibung

In dieser Webanwendung wird das vorhersagen von Wörtern in einem Fließtext dargestellt. Der Nutzer kann einen Text eingeben. Ab dem fünften Wort werden Worte vorgeschlagen aus denen der Nutzer wählen kann oder er schreibt einfach weiter. Bei jedem Drücken der Space-Taste werden neue worte vorgeschlagen. Das Model wurde durch den Text "Wasserscheiden" von Alfred DeMichele trainiert. Dieses Werk hat ca 1 mio Worte von denen gut 20.000 einzigartig sind.
Das Model wurde so trainiert, dass es aus den letzten fünf Worten die nächsten Worte vorhersagt.
